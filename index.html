  • CRUD lengkap data buku, rak fisik, kategori, penulis, dan penerbit.
  • Log audit (activity log) menyimpan aktivitas sistem & IP Address.
  • Ekspor laporan data peminjaman ke format JSON, PDF, dan Excel.

STKI Mie Gacoan

STKI Mie Gacoan — Sistem Temu Kembali Ulasan

Sistem temu kembali informasi ulasan restoran berbasis Vector Space Model (VSM) dan NLP Bahasa Indonesia untuk mengatasi ulasan slang dan salah ketik.

Fitur Utama:

  • Mesin pencarian ulasan terurut menggunakan pembobotan TF-IDF dan Cosine Similarity.
  • Pipeline NLP: Tokenisasi, Case Folding, Stopword Removal, dan Stemming Sastrawi.
  • Normalisasi slang otomatis dengan kamus internal (slang mapping).
  • Daftar kata dilindungi (Protected Words) untuk menjaga nama entitas/brand.
  • Visualisasi detail setiap langkah preprocessing teks pada dokumen.
  • Tabel interaktif ekstraksi fitur berisi nilai frekuensi kata (dfi) dan matriks TF-IDF.
  • Klasifikasi sentimen (Positif, Netral, Negatif) otomatis berdasarkan rating ulasan.
Python (Flask) MySQL Sastrawi (NLP) JavaScript
